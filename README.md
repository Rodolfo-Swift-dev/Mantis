# Mantis

-----------
Mantis es una App gratuita para iOS, que se ha desarrollado para solucionar el problema de tener igual nameUser **nameUser** en distintas plataformas. Pensando en ayudar al usuario a unificar y sincronizar la busqueda y consulta de NameUser para distintas cuentas en las distintas plataformas y redes sociales. (ante todo quiero que sea una App gratuita y en constante evolución).

### ¿De qué forma utilizamos Mantis?
-----------

* Consulta de nombre.
A traves de un campo de texto el usuario ingresa el nombre a consultar para verificar la disponobilidad en las distintas plataformas y redes sociales.
cabe señalar que el texto recibido por la aplicacion pueden ser alfanumericos, mayuscula, minuscula, simbolos y espacios pero al aplicar expresiones regulares solo consideramos el texto alfanumerico, mayuscula y minuscula.

* Darle al boton **Search** para buscar, el cual nos llevara a una nueva vista el cual me representara la respuesta.

* Estando en la vista de respuesta, darle al boton **Back** para volver a la vista principal y ejecutar una nueva busqueda.

Interfaz adaptada para iPAD y iOS con orientacion:
* Portrait

### Requisitos mínimos
-----------

* SWIFT 5.0
* XCODE 14.0
* iOS 13.0

-----------
